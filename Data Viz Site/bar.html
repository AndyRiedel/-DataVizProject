Enter Your Data

Enter labels and values below, delimited by commas (e.g. "A, B, C" labels for "1, 2, 3" values). The number of entered labels must equal the number of entered values for the chart to draw.

To upload a file, select a comma-seperated values file (.csv) from your computer. An example is provided in the site directory.

  • The first row will be assumed to be header information, and will be excluded.
  • Align your x-values in the first column
  • Align your corresponding y-values in the second column
  • The program will sort this data by string or number in the x-axis column (column 1)
